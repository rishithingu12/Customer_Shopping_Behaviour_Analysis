# Customer_Shopping_Behaviour_Analysis
This project analyzes customer shopping behavior using 3,900 transactions to uncover spending patterns and customer segments. Data was cleaned using Python and analyzed with SQL. A Power BI dashboard visualizes key insights, helping drive better business decisions and customer strategies.

📊 Customer Shopping Behavior Analysis

📌 Overview

This project focuses on analyzing customer shopping behavior to uncover meaningful insights from transactional data. It aims to understand spending patterns, customer preferences, and segmentation to support data-driven business decisions.

📂 Dataset
~3,900 transaction records

Includes customer demographics, purchase details, and shopping behavior

Features: Age, Gender, Category, Purchase Amount, Discounts, Subscription Status, Ratings, etc.

Minor missing values handled during data cleaning

🛠️ Tools & Technologies
Python (Pandas, NumPy) – Data loading, cleaning, and EDA

SQL (PostgreSQL / MySQL / SQL Server) – Data analysis

Power BI – Dashboard and visualization

Gamma – Presentation (PPT) creation

🔍 Project Steps
Data Loading – Imported dataset using Python

Data Cleaning – Handled missing values and standardized columns

EDA (Exploratory Data Analysis) – Analyzed trends and patterns

Feature Engineering – Created new columns like age groups

SQL Analysis – Ran queries to extract business insights

Visualization – Built an interactive Power BI dashboard

Reporting – Created detailed report and presentation

📊 Dashboard
The Power BI dashboard provides:

Revenue analysis by customer segments

Product performance insights

Customer behavior trends

Subscription and discount analysis

📈 Results & Insights
Identified high-spending customer segments

Analyzed impact of discounts on revenue

Discovered top-performing products

Segmented customers into New, Returning, and Loyal

Provided actionable business recommendations

🚀 How to Run

Load the dataset in Python (Jupyter Notebook)

Perform data cleaning and EDA

Connect to SQL database and run queries

Import processed data into Power BI

Build dashboard and analyze insights

✨ This project demonstrates end-to-end data analytics workflow, from data processing to visualization and business insights.
